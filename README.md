This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

It is an MVP web application designed to help users discover fishing spots and related information in
Turkey. The platform provides an interactive map and filtering system to explore different fish species by
region and season. It also includes species details, seasonal availability, and equipment
recommendations, offering a user-friendly and responsive design optimized for both desktop and mobile
devices.

# Technologies: Node.js Ecosystem, Next.js, TypeScript, Express, MongoDB
- Cloudflare R2 (Object Storage)
-  Vercel and Render (Deployment)
- Uptime Robot (Preventing cold start)
- Socket.IO (Real time development)
- Auth0 (Authentication and authorization)
